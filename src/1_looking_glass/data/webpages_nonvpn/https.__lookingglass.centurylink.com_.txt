* BGP (Border Gateway Protocol) allows you to view routing table entries for a specified prefix for a selected site.<br>
* PING allows you to view round trip time between a selected site and a specified IP address or hostname.<br>
* TRACEROUTE allows you to view all enabled router hops encountered in the path between a selected site and a specified IP address or hostname.<br>
