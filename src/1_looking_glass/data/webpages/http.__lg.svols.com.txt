<h3 class="wp-block-heading has-text-align-left has-azcom-black-color has-text-color">REMOTE IX (PIPEX)</h3>



<p class="has-azcom-black-color has-text-color">Услуга Remote IX характеризует прямое включение к Internet Exchange – точке обмена трафиком.</p>



<p class="has-azcom-black-color has-text-color">Услуга Remote IX включает в себя Аренду канала до точки обмена трафиком, плюс дает возможность обмена маршрутной информацией с её участниками по протоколу BGP.</p>



<p class="has-azcom-black-color has-text-color">Поскольку стоимость данной услуги значительно ниже, чем IP-Транзит, то пользование данной услугой позволяет подключаемому Интернет-провайдеру увеличить скорость соединения в своих тарифных планах, что позитивно сказывается на росте абонентской базы, а также получить экономию средств, сэкономив на полосе канала по услуге IP- Транзит.</p>

<h3 class="wp-block-heading has-text-align-left has-white-color has-text-color">IP-ТРАНЗИТ</h3>



<p class="has-white-color has-text-color">Услуга IP-транзит служит для обеспечения глобальной связности каждого с каждым, это именно то, чего не хватает при пользовании только одной услугой Remote IX (Ссылка).</p>



<p class="has-white-color has-text-color">В услугу IP-Транзит входит предоставление доступа как ко всему сегменту сети Интернет по протоколу BGP, так и его части, используя при этом BGP Community либо Default Route.</p>



<p class="has-white-color has-text-color">Мы обеспечиваем круглосуточную техническую поддержку клиента, проводим оптимизацию необходимого маршрута доставки контент-трафика, а также оказываем помощь в настройке оборудования. 			sibling,
			skipLinkTargetID,
			skipLink;

		// Early exit if a skip-link target can't be located.
		if ( ! skipLinkTarget ) {
			return;
		}

		// Get the site wrapper.
		// The skip-link will be injected in the beginning of it.
		sibling = document.querySelector( '.wp-site-blocks' );

		// Early exit if the root element was not found.
		if ( ! sibling ) {
			return;
		}

		// Get the skip-link target's ID, and generate one if it doesn't exist.
		skipLinkTargetID = skipLinkTarget.id;
		if ( ! skipLinkTargetID ) {
			skipLinkTargetID = 'wp--skip-link--target';
			skipLinkTarget.id = skipLinkTargetID;
		}

		// Create the skip link.
		skipLink = document.createElement( 'a' );
		skipLink.classList.add( 'skip-link', 'screen-reader-text' );
		skipLink.href = '#' + skipLinkTargetID;
		skipLink.innerHTML = 'Перейти к содержимому';

		// Inject the skip link.
		sibling.parentElement.insertBefore( skipLink, sibling );
	}() );
